# MedicalCard
## Installation guide

> Download the repo `gh repo clone desmeach/medicalCard`
>
> Get into `cd medicalCard`
>
> Execute installation `composer install`
>
> Create file **db_config.php** with database info
>
>
> `<?php`
>
> `$dbHost = 'change-me';`
>
> `$dbName = 'change-me';`
>
> `$dbUser = 'change-me';`
>
> `$dbPass = 'change-me';`
